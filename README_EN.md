# EMHentai

**English** | [中文](README.md)

## Introduction

A simple, lightweight, fast, pure-Swift E-Hentai iOS client.

## Features

- [x] Browse galleries on E-Hentai
- [x] E-Hentai / ExHentai source switching
- [x] Advanced search (by language, rating, etc.)
- [x] Related tag search
- [x] Browsing history
- [x] Download management
- [x] Background downloading
- [x] Tag translation
- [x] Share functionality
- [x] Account & password / Cookie login
- [x] Filter AI-generated / gore content
- [x] iPhone / iPad support
- [x] Dark mode support
- [x] Portrait / landscape support
- [x] Chinese & English localization
- [x] Compatible up to iOS 26

## Installation

Due to the legal risks associated with E-Hentai, no pre-built IPA is provided.

If you have a Mac, you can install directly via Xcode (no paid developer account required):

1. Make sure your macOS and Xcode are up to date; iPhone/iPad iOS version >= 17.0
2. Download ZIP / git clone
3. Open EMHenTai.xcodeproj
4. Run

## FAQ

**Q: I logged in successfully on the website, but the app still shows "Not logged in"?**

A: E-Hentai may block access from IPs that log in too frequently. Try switching to a different node (US/EU IPs are recommended) and log in again.
